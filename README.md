## 📝 Sobre o Projeto
Este é um sistema de **Questionário Interativo** desenvolvido em C#. O objetivo do projeto foi praticar a manipulação de variáveis globais, contadores de pontuação e o controle de estados de componentes (botões e labels) em tempo de execução.

## 🚀 Funcionalidades
* Sistema de pontuação em tempo real (Acertos vs. Erros).
* Bloqueio de respostas após a escolha para evitar duplicidade.
* Reinicialização do questionário.

## 🛠️ Tecnologias
* Linguagem: C#
* Framework: .NET Framework 4.5
* Interface: Windows Forms
